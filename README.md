PROJECT TITLE
Zendo – Your personal daily planner

RESEARCH PROBLEM
People often struggle to stay focused and organized during the day. Many tasks get forgotten or delayed without a simple way to track them.

MOTIVATION
This project helps users stay organized, focused, and motivated by letting them add tasks, set times, mark them complete, and track daily progress.

CONTROL FLOW
User enters a task and optional time → clicks add
Task appears in the list
User can click task to mark it complete
User can delete task
Progress bar and text update automatically
Tasks are saved in browser memory (localStorage) and reload on page refresh
Optional dark/light mode toggle

IMPLEMENTATION STRATEGY
HTML: Structure of the page (input fields, task list, progress bar)
CSS: Styling, responsive design, dark/light mode
JavaScript:
Add, display, toggle, and delete tasks
Save and load tasks from localStorage
Update progress bar and completion text
Toggle dark/light mode

DEPLYOMENT
Fully functional on GitHub Pages or any local server
Public repository with all code (HTML, CSS, JS)
